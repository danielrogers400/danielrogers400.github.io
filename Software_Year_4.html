Software Development:
Year 4

Fourth year software development was mainly concerned with my Final Year Project. This project is a year-long research project into the field of gesture recognition.

My project is described as:

A Java GUI application to enable users of the Leap Motion gesture recognition sensor to create and record new gestures, and store them in xml files for later use by external applications.

 

So why have I chosen this project?

For years, the power of gesture recognition software and hardware has been steadily increasing. However, the focus of this technolgy has been mainly game-based - think of the PS Eye camera, the XBox Kinect etc. I'm interested to see how this technology can be used in a more general purpose role.

When starting my project, it was going to be restricted to a desktop application - simply using the Leap controller to augment the keyboard and mouse. However I found that the Leap API did not easily allow the user to record new gestures, something which was crucial to my application. As such, I shifted the focus of my project to recording new gestures and allowing them to be used by third parties.
