Software Development:
Year 3

Third year software development was mainly concerned with enhancing my knowledge of Java and C++. My Java classes focussed on developing Graphical user Interfaces (GUIs) and my C++ classes were concerned with concepts such as memory management and Polymorphism.

Shown below is my Java third year project.
For my third year Java project I had to create a GUI demo which implemented the CRUD design pattern (Create, Read, Update, Delete). I achieved this by creating a GUI which stored information on the price of HDD storage space over the last 40 years. The application serialises data in order to save it. The source can be viewed and downloaded at https://bitbucket.org/danielrogers/java-third-year-project
